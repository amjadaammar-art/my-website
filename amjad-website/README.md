## How to host on GitHub Pages (free)

1. Create a GitHub account at github.com if you don't have one.
2. Click **+** (top right) → **New repository**.
   - Name it `amjad-website` (or anything you like).
   - For the address `username.github.io` (no repo name in the URL), name the repository exactly `YOURUSERNAME.github.io`.
   - Keep it **Public**, click **Create repository**.
3. On the repository page, click **uploading an existing file**, drag **all the files in this folder** in, and click **Commit changes**.
4. Go to **Settings → Pages** (left sidebar).
5. Under "Branch", select **main**, folder **/ (root)**, and click **Save**.
6. Wait 1–2 minutes. Your site will be live at:
   - `https://YOURUSERNAME.github.io/amjad-website/` (regular repo), or
   - `https://YOURUSERNAME.github.io/` (if you named the repo `YOURUSERNAME.github.io`)

## Updating the site later

Replace any file in the repository (GitHub → the file → the pencil icon or re-upload) and commit. The live site updates automatically in a minute or two.
